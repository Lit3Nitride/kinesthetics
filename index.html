Installation

To install Kinesthetics using npm, simply run npm i kinesthetics

Usage

To compile a .kst file, change the directory to the kinesthetics folder and run node kinesthetics.js path/to/presentation.kst.
A file named presentation.html will be created in the folder path/to.

To test or load a presentation live – that is, for all changes to be reflected once you refresh the page – place the file {name}.kst into the folder slides/{name}/{name}.kst (where slides should be in the same folder as kinesthetics.js), run npm start, then visit http://localhost:3000/slides/${name}
